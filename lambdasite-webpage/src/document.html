Lambdasite

The repository for this tool is at: https://github.com/robmoore-i/lambdasite

It is a few scripts and supporting resources which make it easy for me to create very cheaply hosted documents on the internet using AWS Lambda. This is essentially a very shitty version of Claudia.js which was shown to me by Gojko Adzic at CITCON in May 2018. It was interesting to learn about all the API gateway stuff that goes into getting a live lambda.

For example, this web page was created like this: 

python3 create-page.py lambdasite
cd lambdasite-webpage
* edit src/document.html *
./update-lambda.sh
I am using this collection of webpages to cheaply upload my thoughts to the internet, as though enough people didn't do that already.
