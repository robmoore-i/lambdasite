Lambdasite

What's this?

It's a few scripts and supporting resources which make it easy for me to create cheaply hosted documents on the internet using AWS Lambda. This can be done with a much better called Claudia.js which was shown to me by Gojko Adzic at CITCON in May 2018. It was interesting to learn about all the API gateway stuff that goes into getting a live lambda.

The repository for this tool is at: https://github.com/robmoore-i/lambdasite 

rob$ python3 create-page.py lambdasite
rob$ cd lambdasite-webpage
rob$ * edit src/document.html *
rob$ ./update-lambda.sh

Why?

I am using this collection of webpages to cheaply upload my thoughts to the internet, as though enough people didn't do that already.
